Laytime Calculation Equations
Equation 1: Laytime Allowed (Hours)
text

Laytime Allowed (hours) = (Partial Cargo Quantity ÷ Daily Discharge Rate) × 24

Example:
= (38,460.52 ÷ 5,000) × 24
= 7.692104 × 24
= 184.61 hours
= 7 days 16 hours 37 minutes

Equation 2: Total Laytime Used (Hours)
text

Laytime Used (hours) = Σ [ (End Time - Start Time) × Factor ] 
                       for all events where Excluded = FALSE

Example from your data:
Event: 20-JAN 08:00 to 24:00 = 16 hours × 0.6 = 9.60 hours
Event: 21-JAN 00:00 to 24:00 = 24 hours × 0.6 = 14.40 hours
...
Total Used = 494.35 hours (your calculation shows 37 days 2 hours 10 min = 890.17 hours?)

Note: Your file shows discrepancy. Need to verify if used time includes all events or only partial cargo events.
Equation 3: Demurrage Calculation
text

IF Laytime Used > Laytime Allowed THEN
    Demurrage Days = (Laytime Used - Laytime Allowed) ÷ 24
    Demurrage Amount = Demurrage Days × Daily Demurrage Rate
ELSE
    Despatch Days = (Laytime Allowed - Laytime Used) ÷ 24
    Despatch Amount = Despatch Days × Daily Despatch Rate (usually 50% of demurrage)
END IF

Example from your Laytime PDF:
Laytime Used = 37 days 2 hours 10 min = 890.17 hours
Laytime Allowed = 12 days 19 hours 30 min = 307.50 hours
Difference = 582.67 hours = 24.28 days
Demurrage = 24.28 × $20,000 = $485,555.56

PART 4: EXCLUSION RULES & EQUATIONS
Rule 1: Weekend Exclusion (FHEX - Friday 17:00 to Sunday 08:00)
text

IF (Day = Friday AND Time ≥ 17:00) OR
   (Day = Saturday) OR
   (Day = Sunday AND Time < 08:00) THEN
    Factor = 0 (No time counts)
END IF

Rule 2: Holiday Exclusion
text

IF Date IN Holidays_Table THEN
    Factor = 0 (No time counts)
    // Some holidays also affect NOR acceptance
END IF

Rule 3: Pre/Post Holiday Exclusion
text

IF (Day before holiday AND Time ≥ 17:00) OR
   (Day after holiday AND Time < 08:00) THEN
    Factor = 0 (No time counts)
END IF

Rule 4: Vessel Operations (Shifting/Turning)
text

IF Narration CONTAINS ("shifting" OR "turning" OR "awaiting barge") THEN
    Factor = 0 (Unless specifically negotiated)
END IF

Rule 5: Factor Application
text

Counting Hours = (End Time - Start Time) × Factor

Where:
Factor = 1.0 → Full time counts (on demurrage)
Factor = 0.6 → Partial time counts (working but interrupted)
Factor = 0.0 → No time counts (excluded period)

EQUATION 2: Check If Time Should Count
text

FUNCTION is_time_counting(datetime, voyage, holidays):
    
    # Rule 1: Weekend check (FHEX - Fri 17:00 to Sun 08:00)
    weekday = datetime.weekday()  # Monday=0, Sunday=6
    hour = datetime.hour
    minute = datetime.minute
    
    IF voyage.week_end_day == "Friday" AND voyage.week_end_time == "17:00":
        IF weekday == 4 AND (hour >= 17 OR (hour == 17 AND minute >= 0)):
            RETURN FALSE  # Friday after 17:00
    
    IF voyage.week_start_day == "Sunday" AND voyage.week_start_time == "08:00":
        IF weekday == 5:  # Saturday
            RETURN FALSE
        IF weekday == 6 AND hour < 8:  # Sunday before 08:00
            RETURN FALSE
    
    # Rule 2: Holiday check
    FOR each holiday IN holidays:
        IF datetime.date() == holiday.holiday_date:
            IF holiday.full_day_exclusion:
                RETURN FALSE
        IF holiday.pre_holiday_exclude:
            previous_day = datetime.date() - timedelta(days=1)
            IF previous_day == holiday.holiday_date:
                IF hour >= 17:
                    RETURN FALSE
        IF holiday.post_holiday_exclude:
            next_day = datetime.date() + timedelta(days=1)
            IF next_day == holiday.holiday_date:
                IF hour < 8:
                    RETURN FALSE
    
    RETURN TRUE

EQUATION 3: Calculate Used Laytime
text

FUNCTION calculate_used_hours(voyage_id, voyage, events, holidays):
    total_used_hours = 0
    current_time = voyage.nor_accepted
    
    FOR each event IN events (sorted by event_date, start_time):
        event_start = datetime.combine(event.event_date, event.start_time)
        event_end = datetime.combine(event.event_date, event.end_time)
        
        # Only count from NOR acceptance onward
        IF event_start < voyage.nor_accepted:
            event_start = voyage.nor_accepted
        
        # Calculate duration in hours
        duration_hours = (event_end - event_start).total_seconds() / 3600
        
        # Count only counting portions
        counting_hours = 0
        time_step = timedelta(minutes=15)  # Check every 15 minutes
        
        current = event_start
        WHILE current < event_end:
            IF is_time_counting(current, voyage, holidays):
                IF event.factor > 0:
                    counting_hours += (time_step.total_seconds() / 3600) * event.factor
            current += time_step
        
        total_used_hours += counting_hours
    
    RETURN total_used_hours

EQUATION 4: Calculate Demurrage/Despatch
text

FUNCTION calculate_demurrage(allowed_hours, used_hours, demurrage_rate, despatch_rate):
    difference_hours = used_hours - allowed_hours
    
    IF difference_hours > 0:
        # Demurrage scenario
        demurrage_days = difference_hours / 24
        amount = demurrage_days * demurrage_rate
        return {
            "status": "DEMURRAGE",
            "difference_hours": difference_hours,
            "days": demurrage_days,
            "amount": amount,
            "rate_used": demurrage_rate
        }
    ELSE:
        # Despatch scenario (early completion)
        despatch_days = abs(difference_hours) / 24
        amount = despatch_days * despatch_rate
        return {
            "status": "DESPATCH",
            "difference_hours": difference_hours,
            "days": despatch_days,
            "amount": amount,
            "rate_used": despatch_rate
        }

EQUATION 5: Factor Application Logic
text

FUNCTION apply_factor(event_narration, default_factor):
    narration_lower = event_narration.lower()
    
    # Full counting scenarios (Factor = 1.0)
    IF "on demurrage" IN narration_lower:
        RETURN 1.0
    IF "discharging completed" IN narration_lower:
        RETURN 1.0
    
    # Partial counting scenarios (Factor = 0.6)
    IF "discharging operation continued" IN narration_lower:
        RETURN 0.6
    IF "discharging operation resumed" IN narration_lower:
        RETURN 0.6
    
    # Zero counting scenarios (Factor = 0)
    IF "awaiting for lighter vessel" IN narration_lower:
        RETURN 0
    IF "vessel turning" IN narration_lower:
        RETURN 0
    IF "vessel shifting" IN narration_lower:
        RETURN 0
    IF "suspended due to" IN narration_lower:
        RETURN 0
    IF "weekly holiday" IN narration_lower:
        RETURN 0
    IF "ramadan sehri break" IN narration_lower:
        RETURN 0
    IF "ramadan iftar break" IN narration_lower:
        RETURN 0
    IF "waiting for barge" IN narration_lower:
        RETURN 0
    IF "crane under maintenance" IN narration_lower:
        RETURN 0
    
    RETURN default_factor

from datetime import datetime, timedelta

def calculate_allowed_laytime(partial_cargo_mt, discharge_rate_mt):
    """Equation 1: Calculate allowed laytime in hours"""
    allowed_hours = (partial_cargo_mt / discharge_rate_mt) * 24
    return allowed_hours

def is_excluded_period(dt, voyage, holidays):
    """Equation 2: Check if time period should count"""
    # Weekend check
    weekday = dt.weekday()
    hour = dt.hour
    
    # Friday after 17:00 (weekend starts)
    if voyage.week_end_day == "Friday":
        if weekday == 4 and hour >= 17:
            return True
    
    # Saturday all day
    if weekday == 5:
        return True
    
    # Sunday before 08:00
    if voyage.week_start_day == "Sunday":
        if weekday == 6 and hour < 8:
            return True
    
    # Holiday check
    for holiday in holidays:
        if dt.date() == holiday.holiday_date:
            if holiday.full_day_exclusion:
                return True
        if holiday.pre_holiday_exclude:
            if dt.date() == holiday.holiday_date - timedelta(days=1):
                if hour >= 17:
                    return True
        if holiday.post_holiday_exclude:
            if dt.date() == holiday.holiday_date + timedelta(days=1):
                if hour < 8:
                    return True
    
    return False

def calculate_used_laytime(voyage, events, holidays):
    """Equation 3: Calculate total used laytime"""
    total_hours = 0.0
    
    for event in events:
        start = datetime.combine(event.event_date, event.start_time)
        end = datetime.combine(event.event_date, event.end_time)
        
        # Start counting from NOR acceptance
        if start < voyage.nor_accepted:
            start = voyage.nor_accepted
        
        # Calculate in 15-minute increments
        current = start
        while current < end:
            if not is_excluded_period(current, voyage, holidays):
                # Count 15 minutes with factor
                total_hours += (0.25 * event.factor)
            current += timedelta(minutes=15)
    
    return total_hours

def calculate_demurrage(allowed_hours, used_hours, demurrage_rate, despatch_rate):
    """Equation 4: Calculate demurrage or despatch"""
    diff_hours = used_hours - allowed_hours
    
    if diff_hours > 0:
        days = diff_hours / 24
        amount = days * demurrage_rate
        return {"type": "DEMURRAGE", "days": days, "amount": amount}
    else:
        days = abs(diff_hours) / 24
        amount = days * despatch_rate
        return {"type": "DESPATCH", "days": days, "amount": amount}

PART 5: ALGORITHM PSEUDO-CODE
ALGORITHM 1: MAIN CALCULATION ENGINE
text

ALGORITHM CalculateLaytimeDemurrage
INPUT: voyage_id
OUTPUT: calculation_results

BEGIN
    // Load data
    voyage ← DB.GetVoyage(voyage_id)
    events ← DB.GetEvents(voyage_id, sort by date, start_time)
    holidays ← DB.GetHolidays(voyage.nor_tendered.year to voyage.completion.year)
    
    // Step 1: Calculate allowed laytime
    allowed_hours ← (voyage.partial_cargo_mt / voyage.discharge_rate_mt) * 24
    
    // Step 2: Initialize tracking variables
    total_used_hours ← 0
    on_demurrage ← FALSE
    demurrage_start_time ← NULL
    
    // Step 3: Process each event
    FOR EACH event IN events:
        
        event_start ← DateTime(event.event_date, event.start_time)
        event_end ← DateTime(event.event_date, event.end_time)
        
        // Adjust start time if before NOR acceptance
        IF event_start < voyage.nor_accepted THEN
            event_start ← voyage.nor_accepted
        END IF
        
        // Process in 15-minute intervals
        current_time ← event_start
        WHILE current_time < event_end:
            
            // Determine if this interval counts
            is_excluded ← CheckExclusions(current_time, voyage, holidays)
            
            IF NOT is_excluded THEN
                // Time counts
                effective_factor ← event.factor
                
                // When on demurrage, factor becomes 1.0
                IF on_demurrage THEN
                    effective_factor ← 1.0
                END IF
                
                // Add 15 minutes (0.25 hours) with factor
                total_used_hours ← total_used_hours + (0.25 * effective_factor)
            END IF
            
            // Check if demurrage started
            IF NOT on_demurrage AND total_used_hours > allowed_hours THEN
                on_demurrage ← TRUE
                demurrage_start_time ← current_time
            END IF
            
            current_time ← current_time + 15 minutes
        END WHILE
    END FOR
    
    // Step 4: Calculate demurrage/despatch
    diff_hours ← total_used_hours - allowed_hours
    
    IF diff_hours > 0 THEN
        demurrage_days ← diff_hours / 24
        amount ← demurrage_days * voyage.demurrage_rate
        result_type ← "DEMURRAGE"
    ELSE
        despatch_days ← ABS(diff_hours) / 24
        amount ← despatch_days * voyage.despatch_rate
        result_type ← "DESPATCH"
    END IF
    
    // Step 5: Format and return results
    RETURN {
        allowed: FormatTime(allowed_hours),
        used: FormatTime(total_used_hours),
        difference: FormatTime(ABS(diff_hours)),
        type: result_type,
        days: (diff_hours > 0 ? demurrage_days : despatch_days),
        amount: amount,
        on_demurrage_from: demurrage_start_time
    }
END

ALGORITHM CheckExclusions
INPUT: check_datetime, voyage, holidays
OUTPUT: is_excluded (BOOLEAN)

BEGIN
    weekday ← check_datetime.weekday()  // 0=Mon, 6=Sun
    hour ← check_datetime.hour
    minute ← check_datetime.minute
    time_value ← hour + (minute / 60)
    
    // Rule 1: Weekend check (FHEX)
    // Friday 17:00 to Sunday 08:00
    IF voyage.week_end_day = "Friday" AND voyage.week_end_time = "17:00" THEN
        // Friday after 17:00
        IF weekday = 4 AND time_value >= 17.0 THEN
            RETURN TRUE
        END IF
        
        // Saturday all day
        IF weekday = 5 THEN
            RETURN TRUE
        END IF
        
        // Sunday before 08:00
        IF weekday = 6 AND time_value < 8.0 THEN
            RETURN TRUE
        END IF
    END IF
    
    // Rule 2: Holiday check
    FOR EACH holiday IN holidays:
        // Full day exclusion
        IF holiday.full_day_exclusion AND 
           check_datetime.date() = holiday.holiday_date THEN
            RETURN TRUE
        END IF
        
        // Pre-holiday exclusion (17:00 onward)
        IF holiday.pre_holiday_exclude AND
           check_datetime.date() = (holiday.holiday_date - 1 day) AND
           time_value >= 17.0 THEN
            RETURN TRUE
        END IF
        
        // Post-holiday exclusion (until 08:00)
        IF holiday.post_holiday_exclude AND
           check_datetime.date() = (holiday.holiday_date + 1 day) AND
           time_value < 8.0 THEN
            RETURN TRUE
        END IF
    END FOR
    
    // If no rules matched, time is not excluded
    RETURN FALSE
END

ALGORITHM 3: AUTO-FACTOR DETECTION
text

ALGORITHM DetectFactorFromNarration
INPUT: narration_text (STRING)
OUTPUT: factor (FLOAT)

BEGIN
    text ← narration_text.LOWER()
    
    // Full counting scenarios (Factor 1.0)
    IF text CONTAINS "on demurrage" OR
       text CONTAINS "discharging completed" OR
       text CONTAINS "all cargo discharging completed" THEN
        RETURN 1.0
    END IF
    
    // Partial counting scenarios (Factor 0.6)
    IF text CONTAINS "discharging operation continued" OR
       text CONTAINS "discharging operation resumed" OR
       text CONTAINS "discharging operation commenced" THEN
        
        // Check for gang idle information
        IF text CONTAINS "1 gang idle" OR
           text CONTAINS "one gang idle" THEN
            RETURN 0.6  // Standard for 1 of 2 gangs
        END IF
        
        RETURN 0.6
    END IF
    
    // Zero counting scenarios (Factor 0.0)
    IF text CONTAINS "awaiting" OR
       text CONTAINS "suspended" OR
       text CONTAINS "turning" OR
       text CONTAINS "shifting" OR
       text CONTAINS "waiting for barge" OR
       text CONTAINS "waiting for lighter" OR
       text CONTAINS "crane under maintenance" OR
       text CONTAINS "ramadan sehri" OR
       text CONTAINS "ramadan iftar" OR
       text CONTAINS "weekly holiday" OR
       text CONTAINS "government holiday" OR
       text CONTAINS "preparation for" THEN
        RETURN 0.0
    END IF
    
    // Default factor (user must specify)
    RETURN NULL
END

ALGORITHM ImportSOFEvents
INPUT: file_path (CSV/Excel), voyage_id
OUTPUT: import_summary

BEGIN
    data ← ReadFile(file_path)
    events_imported ← 0
    errors ← []
    
    FOR EACH row IN data:
        
        // Parse row data
        event_date ← ParseDate(row.date)
        start_time ← ParseTime(row.from)
        end_time ← ParseTime(row.to)
        narration ← row.remarks
        
        // Validate times
        IF start_time >= end_time THEN
            errors.ADD("Invalid time range for row " + row.number)
            CONTINUE
        END IF
        
        // Auto-detect factor
        factor ← DetectFactorFromNarration(narration)
        
        IF factor IS NULL THEN
            // Prompt user for factor or use default 0.6
            factor ← 0.6
            WARNING("Factor not detected for: " + narration)
        END IF
        
        // Check for overlap with existing events
        existing ← DB.GetEventsByTimeRange(voyage_id, event_date, start_time, end_time)
        
        IF existing.COUNT > 0 THEN
            errors.ADD("Overlap detected with event " + existing[0].id)
            CONTINUE
        END IF
        
        // Save event
        DB.Insert("sof_events", {
            voyage_id: voyage_id,
            event_date: event_date,
            start_time: start_time,
            end_time: end_time,
            factor: factor,
            narration: narration,
            created_at: CURRENT_TIMESTAMP
        })
        
        events_imported ← events_imported + 1
    END FOR
    
    // Recalculate voyage
    IF events_imported > 0 THEN
        TriggerRecalculation(voyage_id)
    END IF
    
    RETURN {
        imported: events_imported,
        errors: errors
    }
END
